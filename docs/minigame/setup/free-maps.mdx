## Pre-made Setups/Maps {#premade-setup}

In this section, you can download some additional maps for your server.
:::tip
Every map contains a configured setup which you can easily drag and drop. The arena is already configured on arenas.yml so you don't have to care! Make sure to read the README file on the folder you downloaded!
:::
:::caution
If you import multiple setups you indeed need to manually add the arena configuration section to your orginal arenas.yml and should not overwrite the file!
:::
